Global Navbar Integration

This page demonstrates how to integrate the global navbar into any page with just one script tag.

Easy Integration

Simply include the integration script and the global navbar will be automatically added to your page.

Responsive Design

The navbar automatically adapts to different screen sizes with a mobile-friendly hamburger menu.

Theme Support

Built-in dark/light theme support that automatically syncs with your page's theme.

Global Search

Integrated search functionality that searches across all content types in your application.

Notifications

Real-time notification system with badge counts and dropdown notifications.

User Menu

Complete user management with authentication, profile access, and logout functionality.

How to Integrate

Add this single script tag to any HTML page:

<!-- Include Font Awesome for icons -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Include global navbar integration script -->
<script src="/js/navbar-integration.js"></script>

<!-- Add top padding to account for fixed navbar -->
<style>
    .page-content {
        padding-top: 70px; /* Height of navbar */
    }
</style>

Advanced Integration

For more control, you can use the JavaScript API:

// Wait for navbar to be ready
document.addEventListener('DOMContentLoaded', () => {
    if (window.globalNavbar) {
        // Access navbar instance
        console.log('Navbar is ready:', window.globalNavbar);
        
        // Listen for theme changes
        document.body.addEventListener('classChange', (e) => {
            if (e.detail.classList.contains('dark-mode')) {
                console.log('Theme changed to dark mode');
            }
        });
    }
});
