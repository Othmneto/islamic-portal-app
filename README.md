# Translator Backend

This is the backend for the Shaikh Voice Translator & Prayer Times app.

## Features

- Multilingual translation API
- Islamic resources (Duas, Hadith, country/Islamic data)
- Text-to-speech and voice support
- City and country-specific endpoints
- Audio upload and response features

## Getting Started

### Prerequisites

- Node.js (v18.x or above recommended)
